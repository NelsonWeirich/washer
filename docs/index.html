Washer Library

Introduction

This user manual describes the washer software application, a suite of common functions for use on your washer machine.

As funções de bibliotecas são divididas em cinco diretórios:

  • adc_lib
  • display_lib
  • pin_lib
  • pt_lib
  • usart_lib

As funções, assim como a organização do diretório principal podem ser encontradas em file
